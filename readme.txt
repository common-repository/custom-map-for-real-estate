== Installation ==
1. Custom Map for Real Estate can be installed directly through your WordPress
Plugins dashboard.
1. Click "Add New" and Search for "Custom Map for Real Estate"
1. Install and Activate

== Used ==
Copy shortcode to view your custom map: [map4re id="YOUR_ID"]
